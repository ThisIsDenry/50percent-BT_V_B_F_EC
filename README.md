# 50percent-BT_V_B_F_EC
Searches for structure seeds with buried treasure, village, bastion, fortress, and an end city near (0,0).
To run, first set up a few parameters in the "options.txt" file in the "app" folder.
The first line of the text file is the structure seed the finder begins on.
The second line is the maximum number of seeds you want to find per session.
The third line is the number of threads you want to use. Don't set this to more than you computer can handle.
Once "options.txt" is set up, navigate back to your master folder and open command prompt.
Run ".\gradlew run" (on Windows).
Run "gradlew run" (on Mac).
